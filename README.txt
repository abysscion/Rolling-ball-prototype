После открытия проекта на сцене по умолчанию "SampleScene" находятся 4 объекта:
1) "[UI]" - содержит в себе элементы интерфейса
2) "[Environment] - содержит в себе предметы окружения. Свет, объект-холдер для скрипта генератора дороги и стартовая площадка 3х3 тайла размером.
3) @GameController - объект-холдер для синглтона геймконтроллера. Позволяет выставить уровень сложности (до запуска сцены) в инспекторе в поле Difficulty, а также выбрать какое из правил будет применятся генерации кристаллов. Если "Crystal Should Spawn Randomly" выставлен как true, генерация кристаллов будет происходить случайным образом, в противном случае по порядку.
4) Player - содержит в себе скрипт управления игроком, объект вращающейся сферы и камеру. Позволяет выставить в инспекторе в поле "speed" скорость движения сферы и скорость вращения вокруг своей оси.
